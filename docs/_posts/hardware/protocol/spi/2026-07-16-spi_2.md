# SPI 协议层详解

## 时钟极性与时钟相位
- 时钟极性：决定 SPI 空闲时的时钟信号是**高电平**（CPOL=1）还是**低电平**（CPOL=0）
- 时钟相位：决定 SPI 总线从 SCK 的**第1个**跳变沿（CPHA=0）还是**第2个**跳变沿（CPHA=1）开始采样数据

### SPI 传输模式
- CPOL与CPHA共同决定SPI的四种模式
