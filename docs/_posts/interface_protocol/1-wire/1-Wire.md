# 1-Wire 单总线协议

> 📝 本文正在编写中...

## 概述

1-Wire（单总线）是由 Dallas Semiconductor（现 Maxim Integrated）开发的串行通信协议，仅需一条数据线即可实现双向通信，同时可通过数据线为从设备供电（寄生供电模式）。

### 特点

1. 单线通信：仅需一条数据线（DQ）+ 地线
2. 支持多点挂载：单总线上可挂载多个从设备
3. 每个从设备有唯一的 64 位 ROM ID
4. 支持寄生供电：从设备可从数据线获取电能

### 典型设备

- DS18B20 温度传感器
- DS2431 EEPROM
- DS2408 8 通道可编程 I/O
